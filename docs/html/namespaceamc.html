Detailed Description

Implementation of the C^0 surface reconstruction of Jiao and Wang, "Reconstructing high-order surfaces for meshing".

Class to serve as abstract generic mesh class.

Implementation of data structure and setup for 3D unstructured mesh.

Classes to reconstruct a C^1 (or C^2) piecewise polynomial (cubic) boundary from a piecewise linear boundary given by a hybrid linear mesh.
