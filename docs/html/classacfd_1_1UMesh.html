void acfd::UMesh::compute_topological ( )
inline

Computes various connectivity data structures for the mesh.

These include (1) Elements surrounding points (esup and esup_p) (2) Points surrounding points (psup) (3) Elements surrounding elements (esuel) (4) Elements surrounding edge (elsed) (5) Edge data structure (intedge) (6) Face data structure (intfac)

NOTE: Currently only works for linear mesh - and psup works only for tetrahedral or hexahedral linear mesh 

Computes, for each face, the elements on either side, the starting node and the ending node of the face. This is stored in intfac.

The orientation of the face is such that the face points towards the element with larger index. NOTE: After the following portion, esuel holds (nelem + face no.) for each ghost cell, instead of -1 as before.
