Detailed Description

Movement of a point-cloud based on radial basis function interpolation of some other points.

Parameters
int_pointsis a list of all interior points to be moved
boun_pointsis the array of boundary points
boundary_motionis nbpoin-by-ndim array - containing displacements corresponding to boundary points.
rbf_chindicates the RBF to use - 0 : C0, 2 : C2, 4 : C4, default : Gaussian
num_stepsis the number of steps in which to break up the movement to perform separately (sequentially)
linear_solverindicates the linear solver to use to solve the RBF equations - "DLU", "CG", "LU"
