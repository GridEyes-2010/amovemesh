Detailed Description

Class HermiteSpline2d constructs one spline curve from all the boundary faces of a mesh which have markers contained in rfl. Currently reconstructs a C^1 boundary. But we need to ensure that out of the two possible tangents for each face, the correct one is chosen for consistency. For this, we calculate normal based on intfac, as face nodes in intfac are ordered to point outwards. NOTE: Make sure the boundaries with markers in rfl are contiguous. CAUTION: There are issues with the functionality of this implmenetation. Spurious loops are created at control points, strangely.
